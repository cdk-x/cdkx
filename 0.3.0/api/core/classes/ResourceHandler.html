    • waitUntilStabilized(
          check: () => Promise<StabilizeStatus>,
          config: StabilizeConfig,

      Poll check until the resource reaches a stable, usable state.

      Returns immediately when check returns { status: 'ready' }. Throws the supplied reason when check returns { status: 'failed' }. Throws StabilizationTimeoutError when config.timeoutMs elapses before a ready response is received.

      Calls check at approximately config.intervalMs intervals.
