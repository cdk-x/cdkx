# @cdk-x/ssh

SSH provider constructs for cdkx. Contains the L1 `ProviderResource` subclasses that represent SSH-managed resources in a cdkx app.

## Role

- Defines SSH resource constructs (e.g. `SshCommand`, `SshFile`) that users place in their construct tree
- Synthesizes each resource to a JSON manifest entry consumed by `@cdk-x/engine`
- Analogous to `@cdk-x/hetzner` in the provider hierarchy

## Relationship with other packages

| Package | Relationship |
|---|---|
| `@cdk-x/core` | Extends `ProviderResource`, uses `Stack`, `App` |
| `@cdk-x/ssh-runtime` | Provides `RUNTIME_CONFIGS` referenced at deploy time |
| `@cdk-x/ssh-sdk` | No direct dependency |
| `@cdk-x/engine` | Reads manifests produced by this package's synthesizer |

## Construct prefix

`Ssh` — e.g. `SshCommand`, `SshFile`.

## Conventions

- L1 constructs are auto-generated by `spec-to-cdkx` in future slices
- All public API exported from `src/index.ts`
